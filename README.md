# Moodify

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## Overview
Moodify is a modern, intuitive mood tracking application designed to help you record, analyze, and improve your emotional wellbeing over time. By logging your mood entries daily, you gain valuable insights and empowering visualizations to better understand your mental health trends.

## Features
- Simple daily mood logging with customizable tags
- Interactive mood calendar and trend charts
- Insights and motivational tips based on your mood history
- Export your data as CSV for personal use
- Responsive design for seamless use on mobile and desktop

## Tech Stack
- Frontend: React.js with Tailwind CSS
- Backend: Node.js with Express
- Database: MongoDB
- Authentication: JWT

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/omgwillmood/moodify.git
   ```
2. Install dependencies:
   ```
   cd moodify
   npm install
   ```
3. Set up environment variables in a `.env` file:
   ```
   MONGO_URI=your_mongodb_connection_string
   JWT_SECRET=your_jwt_secret_key
   ```
4. Start the development server:
   ```
   npm start
   ```

## Usage
- Register or login with your account.
- Add daily mood entries by selecting your mood and optional tags.
- View your mood history with calendar and analytical charts.
- Export your mood data anytime from the settings page.
